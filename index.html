Hidviz

Current version: 0.1.4

About

Upstream URL: https://github.com/ondrejbudai/hidviz

Hidviz is a GUI application for in-depth analysis of USB HID class devices. The 2 main usecases of this application are reverse-engineering existing devices and developing new USB HID devices.

Installation

Fedora

# dnf install hidviz

Arch Linux

# pacaur -S hidviz

Windows

Install from https://github.com/ondrejbudai/hidviz/releases/download/v0.1.4/hidviz-win.zip
